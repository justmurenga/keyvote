## 📐 Image Specifications

### 1. Profile Picture
**Dimensions:** 180x180px minimum (recommended: 500x500px for quality)
**Format:** PNG with transparent background
**File Size:** Under 100KB

**Design Specifications:**
```
Option A: Logo-Only
- myVote Kenya logo centered
- Transparent or white background
- Include small Kenyan flag element (subtle)
- Padding: 20px all sides

Option B: Logo with Circle Background
- Logo in center
- Circular blue gradient background
- White logo or white text
- Modern, clean look
```

**Design Instructions for Graphic Designer:**
1. Create square canvas: 1000x1000px
2. Add logo centered with 60px padding
3. Use primary blue (#1E40AF) as background
4. Add subtle gradient (lighter blue to primary)
5. Include "myVote Kenya" text below logo if space permits
6. Export as PNG with transparency
7. Ensure clarity at 170x170px (Facebook display size)

**Tools You Can Use:**
- Canva: Use "Facebook Profile Picture" template
- Photoshop/GIMP: 1000x1000px canvas
- Online: Remove.bg for background removal

## 🎯 Image Creation Workflows

### Using Canva (Recommended for Beginners)

**Step 1: Set Up Templates**
1. Create Canva account (free)
2. Create "Brand Kit" with myVote colors
3. Upload myVote logo
4. Set brand fonts

**Step 2: Create Post Template**
1. Click "Create a design"
2. Select "Facebook Post" (1200x630px)
3. Add background color/gradient
4. Add text placeholders
5. Add logo placement
6. Save as template

**Step 3: Batch Create Posts**
1. Duplicate template 7 times
2. Customize text for each post
3. Add unique icons/images
4. Download all as JPG
5. Name files: post1.jpg, post2.jpg, etc.

**Canva Pro Tip:**
- Use "Magic Resize" to convert posts to Stories
- Create brand templates for consistency
- Use grids for infographics

---

### Using Photoshop/GIMP (Advanced)

**Template Setup:**
```
1. Create PSD template (1200x630px)
2. Layer structure:
   - Background layer
   - Color overlay layer
   - Content layer
   - Text layer
   - Logo layer
3. Use Smart Objects for easy editing
4. Save as .PSD template
```

**Batch Processing:**
```
1. Create all 7 variations
2. Export for Web (JPG, 80% quality)
3. Max file size: 1MB
```

## 📱 Mobile Optimization

### Text Readability
- Minimum font size: 24px for body text
- Maximum 3-4 lines of text per image
- High contrast (white on dark, dark on light)
- Avoid thin fonts

### Visual Hierarchy
- Largest element: Main message
- Medium: Supporting details
- Smallest: Logo/branding
- Use size to guide attention

### Safe Zones
- Keep important content 100px from edges
- Facebook may crop images on mobile
- Test on actual mobile device

## 💡 Pro Tips

1. **Consistency is Key**
   - Use same fonts across all images
   - Stick to brand colors
   - Maintain similar layouts

2. **Less is More**
   - Don't overcrowd images
   - One main message per image
   - White space is powerful

3. **Optimize for Mobile First**
   - 90% of users on mobile
   - Large, readable text
   - Simple compositions

4. **A/B Test**
   - Try different styles
   - See what resonates
   - Track engagement metrics

5. **Keep Source Files**
   - Save PSD/Canva files
   - Easy to update later
   - Maintain consistency

6. **Batch Create**
   - Create multiple posts at once
   - Use templates for efficiency
   - Schedule in advance
