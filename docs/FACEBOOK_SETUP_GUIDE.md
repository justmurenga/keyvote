### Step 8.3: Install Facebook Pixel on Website

**What is Facebook Pixel?**
A code snippet that tracks website visitors from Facebook ads and posts.

**Benefits:**
- Track conversions
- Retarget website visitors
- Build lookalike audiences
- Measure ad effectiveness

**How to Install:**
1. Go to Facebook Business Manager: business.facebook.com
2. Navigate to Events Manager
3. Click "Connect Data Sources" → "Web"
4. Select "Facebook Pixel"
5. Click "Get Started"
6. Name your pixel: "myVote Kenya Pixel"
7. Enter website URL: www.myvote.ke
8. Choose installation method

**Installation Method (for developers):**
- Copy pixel code
- Add to website <head> section
- Or use integration (Google Tag Manager, etc.)

## 🆘 Troubleshooting Common Issues

### Issue: Can't create username
**Solution:**
- Must have at least 100 page likes first
- Or wait 24 hours after page creation
- Try different variations if taken

### Issue: Posts not reaching followers
**Solution:**
- Facebook prioritizes engagement
- Post when followers are online (check Insights)
- Use engaging content (questions, images, videos)
- Encourage shares and comments
- Consider boosting important posts

### Issue: Too many spam comments
**Solution:**
- Enable moderation (Settings → General → Page Moderation)
- Add common spam words to block list
- Enable profanity filter
- Require post approval before publishing

### Issue: Losing followers
**Solution:**
- Review content quality
- Don't post too frequently
- Engage with community
- Provide consistent value
- Ask for feedback
