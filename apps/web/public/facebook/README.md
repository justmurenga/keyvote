## 📥 How to Convert SVG to PNG/JPG (For Facebook Upload)

Facebook accepts both, but PNG is recommended for the profile picture and JPG for the cover photo.

### Option 1: Online Converter (Easiest - 30 seconds)

1. Go to: **https://cloudconvert.com/svg-to-png**
2. Upload the SVG file
3. Set output dimensions:
   - Profile: **500×500** or **1000×1000**
   - Cover: **1640×624**
4. Click Convert → Download

### Option 2: Use Browser (Free, No Signup)

1. Open the SVG file in Chrome/Edge: drag-drop into browser
2. Right-click → "Save image as..."
3. Or take a screenshot at the size you need

### Option 3: Use Inkscape (Free Desktop Software)

```
1. Download Inkscape: inkscape.org
2. Open the .svg file
3. File → Export → PNG
4. Set DPI to 300 for high quality
5. Click Export
```

### Option 4: Command Line (For Developers)

If you have ImageMagick installed:
```powershell
magick convert -density 300 profile-picture.svg -resize 1000x1000 profile-picture.png
magick convert -density 300 cover-photo.svg -resize 1640x624 cover-photo.jpg
```

## 📋 Quick Reference: Image Specs

### Facebook Profile Picture
- **Display size:** 170×170 (desktop), 128×128 (mobile)
- **Upload size:** 500×500 minimum, 1000×1000 recommended
- **Format:** PNG (transparent) or JPG
- **Max file size:** 100KB recommended
- **Shape:** Will be cropped to circle

### Facebook Cover Photo
- **Display size:** 820×312 (desktop), 640×360 (mobile)
- **Upload size:** 1640×624 recommended
- **Format:** JPG (better for photos) or PNG (better for graphics with text)
- **Max file size:** Under 100KB if possible
- **Safe zone:** Keep important content centered (mobile crops sides)

## 🆘 Troubleshooting

**SVG won't open?**
- Open with browser (Chrome/Edge) or Inkscape

**AI image looks weird?**
- Regenerate (each try gives different result)
- Add to prompt: "high quality, professional, realistic"
- Avoid: "hands holding pen" (AI struggles with hands)

**Text not readable on cover?**
- Test on mobile preview before publishing
- Keep text in center-safe zone
- Use high contrast (white text on dark background)

**File too large for Facebook?**
- Compress at: https://tinypng.com
- Or save as JPG with 80% quality
